# YELPCAMP

### How to get onto local machine 

`git clone https://github.com/arinarjani/yelpcamp yelpcamp` then
`cd yelpcamp`

### How to add packages

`npm install`

### Before running
create a `.env` file with `PORT` and `MONGODB_URI` variables

### Running the app
`npm start`

### Technologies Used
1. Node.js
2. Express JS
3. EJS
4. Passport.js
5. NPM
6. HTML, CSS
